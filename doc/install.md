# Installation

The following packages are required.

- aiida-core
- aiida-vasp
- phonopy
- seekpath
- phono3py (optional)
- aiida-quantumespresso (optional)
- aiida-donothing (optional)
- ALM (optional)

```bash
git clone https://github.com/atztogo/aiida-phonoxpy.git
cd aiida-phonoxpy
pip install -e .
```
